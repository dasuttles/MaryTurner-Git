Known Georgia Lynching Victims

The Mary Turner Project is in the process of developing a searchable database of all known lynchings in the U.S. We hope to complete that database in the upcoming weeks but until then below is a list of documented lynching cases in Georgia.

Please note that this list does not include everyone who has been lynched since many of these crimes were never officially documented. We will be updating the site as our research progresses so please revisit it from time to time. If you have any information about these or other lynchings please contact us.
